# 🏠 House Price Prediction API (MLOps Project)

## 📌 Overview
This project builds an end-to-end Machine Learning pipeline to predict house prices using regression models, MLflow, and FastAPI.

---

## 🚀 Features
- Data cleaning pipeline
- Feature engineering
- Model training (RandomForest)
- Experiment tracking (MLflow)
- Model registry
- FastAPI for serving predictions

---

## 📊 Model Performance
- MAE: ~₹1.26 Lakhs
- R² Score: ~0.70
- Cross-validation R²: ~0.66

---

## 🛠️ Tech Stack
- Python
- Scikit-learn
- MLflow
- FastAPI
- Uvicorn
